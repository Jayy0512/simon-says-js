# Simon Says Game 🎮

A web-based recreation of the classic Simon memory game, built entirely with vanilla web technologies. 

## 📝 Description
This project focuses on core programming logic, DOM manipulation, and event handling. The game generates a random sequence of colors that gets progressively longer. The player must repeat the sequence perfectly to advance to the next level. If the wrong color is clicked, the game ends and displays the highest level achieved.

## ✨ Features
* **Dynamic Sequence Generation:** Uses randomized arrays to build the game's memory sequence.
* **Interactive UI:** Visual feedback for both system-generated sequences and user clicks.
* **State Management:** Tracks current level, user progress, and game over states.

## 🛠️ Tech Stack
* **HTML5:** Page structure and layout.
* **CSS3:** Styling, flexbox positioning, and visual flash effects.
* **JavaScript (ES6):** Core game logic, `setTimeout` functions, array methods, and event listeners.
